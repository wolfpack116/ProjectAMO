- [ ] **브리핑 컨텍스트 레이어 토글** — 비행경로상 위험기상(예: 뇌우 SIGMET) 있을 때만 브리핑 창에 "관련 레이어 보기" 버튼 노출 → 누르면 레이더/낙뢰/SIGMET ON. (자동켜기 아님, 버튼) · [근거](search-feature.md) §6
- [ ] **경로입력 레이어 토글** — 경로 입력 창에서 웨이포인트/항로(ats-route·rnav-route) 켜고 끄는 버튼. 동일 `toggleLayer(id)` 재사용. · §6
- [ ] **위험현상 → 레이어 매핑** — 뇌우=radar+lightning+sigmet 등. 브리핑 기능 소유의 별도 맵(코어 레지스트리 제외). · §6
- [ ] **검색: 자동 레이어 켜기 옵션** — 컨텍스트에서 버튼 대신 자동 ON(후속 옵션). · §6
