항공날씨 데이터를 브리핑 섹션에 매핑

사이트맵 항목을 우리 4개 섹션에 배치했습니다. 데이터 API가 있어 위험강조까지 가능한 것과 차트 이미지뿐이라 링크/썸네일로만 가능한 것을 구분했습니다. 아래 ☐ 항목 중 v1에 넣을 것을 골라주세요.

① 위험 요약
✓ 이미: SIGMET · AIRMET · 공항/급변풍 경보
② 현재 실황 (출발/도착/교체)
✓ 이미: METAR · AMOS
③ 노선·공역 (단면도 링크 = A 확정)
✓ 이미: KIM 바람·기온·구름·착빙 · KTG 난류 · 경로 SIGMET/AIRMET 교차
④ 목적지 예보
✓ 이미: TAF

v1에 추가 후보 (클릭해서 선택)

+

이륙예보 → ② 출발 섹션 (추천)

출발공항 이륙시점 풍향·풍속·기온·기압. API 있음, 가벼움

+

제주공항 급변풍 실황(LIDAR·LLWAS·양배풍) → ①② (추천)

RKPC 출도착 시 한국 특화 윈드시어. 제주 관련 비행에 고가치

+

태풍정보 · 화산재주의보 → ① 위험요약 (추천)

계절성이나 발효 시 최상위 위험. 발효 중일 때만 배지로 노출

+

AIREP(조종사 보고) → ③ 노선

예보 위험을 실측으로 교차검증. 단, 신규 수집 연동 필요(중간 비용)

+

위험기상 알림 → ① 상단 배너

항공기상청 위험기상 알림을 브리핑 최상단에 띄움

+

일출·일몰/박명 · 달빛 → VFR 모드 전용

야간 VFR 판단용. VFR 모드에서만 표시

+

차트 링크형: 종관일기도·난류예측일기도·항공로 연직시계열

정형 데이터 아님 → 썸네일+링크로만. 가볍게 "참고 차트" 묶음
